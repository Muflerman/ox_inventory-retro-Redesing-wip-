> [!IMPORTANT]
> **English:** This is a work-in-progress fork of [ox_inventory](https://github.com/overextended/ox_inventory). It is currently being redesigned to match a specific vintage/classic aesthetic.
>
> **Español:** Este es un fork en desarrollo de [ox_inventory](https://github.com/overextended/ox_inventory). Actualmente está siendo rediseñado para coincidir con una estética vintage/clásica específica.

---

## 🛠 Recent Changes / Cambios Recientes

### English
- **New Visual Identity:** Updated the primary interface to a matte brown theme (`#5d4037`).
- **Enhanced Clarity:** Disabled default screen blur and removed the dark background overlay for a cleaner look.
- **HUD Synergy:** Integrated automatic status card visibility with `clasic-hud`.
- **UI Refinement:** General cleanup of the interface elements for better consistency.

### Español
- **Nueva Identidad Visual:** Se actualizó la interfaz principal a un tema marrón mate (`#5d4037`).
- **Claridad Mejorada:** Se desactivó el desenfoque de pantalla y se eliminó la capa oscura de fondo para un aspecto más limpio.
- **Sinergia con el HUD:** Integración de visibilidad automática de tarjetas de estado con `clasic-hud`.
- **Refinamiento de UI:** Limpieza general de elementos de la interfaz para una mejor consistencia.
